Generate code that will facilitate you to unlock physical features on site if award unlocked from automatic call.

It is useful in cases like

e.g if you want to unlock certain rewards when user reach to level 9.

When user reach to level 9, let's say you want to unlock reward that can increase discount by 1%

When user reach to level 9, auto script will update user account level and unlock rewards. but there should be a section or function which will adjust custom website features too associate with reward. e.g increase %1 discount rate.

Below function Process() will be automatically call when reward unlocked by passing reward id

You have to write custom code for each reward manually so that script can execute at time of awarding reward.

Copy code from below and paste it under /include/events/process_reward_features.php file. If you already add custom script for rewards, then just copy cases which is not yet available in your file.
